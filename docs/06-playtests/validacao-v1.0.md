# Pontos de validação da v1.0

> Origem: Notion / Iconia TCG / Playtest / Pontos de validação da v1.0  
> Sincronizado em: 18/07/2026

## Ritmo

- [ ] Quantos turnos dura uma partida?
- [ ] Em qual turno alguém chega a 20 Iconitos?
- [ ] Cartas caras entram em jogo?
- [ ] Existe recuperação depois de ficar atrás?

## Vitória

- [ ] 20 Iconitos produz duração adequada?
- [ ] Exploração 2 ou 3 acelera demais?
- [ ] Bloquear é suficiente?
- [ ] Há vitória inevitável sem interação?

## Faísca

- [ ] Uma Faísca por turno é adequada?
- [ ] Toda carta como Faísca elimina problemas de recurso?
- [ ] O custo de perder uma carta da mão é relevante?
- [ ] Jogadores ficam sem cartas cedo?
- [ ] A ausência de limite de mão cria problemas?

## Primeiro jogador

- [ ] Pular a primeira compra compensa agir primeiro?
- [ ] A progressão de custos cria vantagem?
- [ ] Explorar primeiro é forte demais?

## Combate

- [ ] Bloqueio 1×1 é intuitivo?
- [ ] Criaturas Cansadas ficam vulneráveis demais?
- [ ] Desafiar aumenta efeito bola de neve?
- [ ] Criaturas Preparadas ficam protegidas demais?
- [ ] Limpar dano no fim do turno é fácil?

## Habilidades

- [ ] Voador possui respostas?
- [ ] Protetor é frustrante?
- [ ] Vigia + Protetor é forte demais?
- [ ] Veloz acelera demais?
- [ ] Feroz tem impacto?
- [ ] Curador encontra alvos?
- [ ] Tóxico torna Vitalidade irrelevante?
- [ ] Qual redação de Resistente funciona melhor?
- [ ] O custo de retorno de Herói deve ser 3?

## Marcadores

- [ ] Limite de 3 é fácil de visualizar?
- [ ] Crianças distinguem Marcador e efeito?
- [ ] Ícones são reconhecíveis?
- [ ] Congelado precisa de lembrete de duração?
- [ ] Quebrado é lembrado sem texto?

## Reciclagem

- [ ] Embaralhar a Reserva alonga demais?
- [ ] Combos podem repetir indefinidamente?
- [ ] Remover possui função suficiente?

## Heróis

- [ ] Retorno direto ao Campo é forte demais?
- [ ] Retorno à mão é mais simples?
- [ ] Retorno conta como Invocar?
- [ ] Efeitos de entrada ativam?
- [ ] Herói retornado entra Cansado apesar de Veloz?
- [ ] O custo deve variar?
