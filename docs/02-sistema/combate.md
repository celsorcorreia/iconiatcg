# Combate

> Origem: Notion / Iconia TCG / Sistema / Combate  
> Sincronizado em: 18/07/2026

## Explorar

1. Escolha uma criatura Preparada.
2. Canse-a.
3. O oponente decide se Bloqueia.
4. Sem Bloqueio, receba Iconitos iguais à Exploração.
5. Com Bloqueio, resolva combate.

Uma criatura Bloqueada não gera Iconitos, mesmo que sobreviva.

## Bloquear

1. Escolha uma criatura Preparada.
2. Canse a bloqueadora.
3. As criaturas causam dano simultaneamente igual ao próprio Poder.
4. Verifique Dano Maior.
5. Envie criaturas Superadas à Reserva.

**Vigia** permite Bloquear mesmo Cansada.

## Desafiar

1. Escolha uma criatura sua Preparada.
2. Canse-a.
3. Escolha uma criatura inimiga Cansada.
4. Cause dano simultâneo.
5. Verifique Dano Maior.
6. Envie criaturas Superadas à Reserva.

Pelas regras básicas, criaturas Preparadas não podem ser Desafiadas.

## Dano Maior

Uma criatura recebe Dano Maior quando o dano acumulado é igual ou maior que sua Vitalidade. Ela é Superada e vai para a Reserva.

Todo dano desaparece no fim do turno.

## Ordem recomendada

1. determinar participantes;
2. aplicar modificadores de Poder;
3. causar dano simultâneo;
4. aplicar prevenção ou redução;
5. verificar Tóxico;
6. verificar Dano Maior;
7. enviar cartas Superadas à Reserva.

> A interação entre Resistente e Tóxico ainda precisa de validação.
