# Invocações e tipos de carta

> Origem: Notion / Iconia TCG / Sistema / Invocações  
> Sincronizado em: 18/07/2026

## Invocar

1. Escolha a carta.
2. Verifique o custo.
3. Apague Faíscas iguais ao custo.
4. Coloque a carta na zona adequada.
5. Resolva seus efeitos.

A carta só pode ser jogada se todo o custo puder ser pago.

## Tipos

| Tipo | Função |
|---|---|
| Criatura | Explora, Bloqueia, Desafia e usa habilidades |
| Magia | Produz efeito único e vai para a Reserva |
| Item | Permanece no Campo ou anexado |
| Lugar | Altera regras gerais; reservado para expansões |

## Entrada

### Criaturas

- entram Preparadas;
- não podem Explorar no turno de entrada;
- podem Explorar ou Desafiar imediatamente com Veloz.

### Magias

Resolvem e vão para a Reserva.

### Itens

Entram no Campo ou são anexados. Quando a criatura anexada sai do Campo, o Item também vai para a Reserva, salvo regra contrária.

## Atributos de criatura

| Informação | Ícone | Função |
|---|---|---|
| Poder | Espada | Dano causado |
| Vitalidade | Coração | Dano suportado |
| Exploração | Cristal/losango | Iconitos obtidos |

O valor de Exploração deve ficar na lateral esquerda da carta.

## Elementos

| Elemento | Cor | Estilo |
|---|---|---|
| Fogo | Vermelho | Dano, impulso, velocidade |
| Água | Azul | Controle, Recuar, Congelado, Espiar |
| Terra | Verde | Defesa, força, resistência |
| Ar | Amarelo | Movimento, Voador, Vigia, Exploração |
