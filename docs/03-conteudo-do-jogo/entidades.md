# Entidades

> Origem: Notion / Iconia TCG / Conteúdo do Jogo / Entidades  
> Sincronizado em: 18/07/2026

**Estado:** reservada para desenvolvimento.

A v1.0 não define Entidade como tipo de carta ou categoria narrativa.

Antes de criar Entidades, definir:

- conceito narrativo;
- tipo de carta;
- permanência no Campo;
- método de Invocação;
- relação com Criaturas e Heróis;
- limite de cópias;
- regras próprias;
- coleção ou expansão de estreia.

Heróis permanecem classificados como Criaturas até decisão explícita.
