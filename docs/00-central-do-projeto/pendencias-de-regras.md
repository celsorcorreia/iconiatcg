# Pendências de regras

> Origem: Notion / Iconia TCG / Registro de Decisões / Pendências de Regras  
> Sincronizado em: 18/07/2026

Este arquivo não estabelece regras finais. Ele registra decisões que precisam ser tomadas e validadas.

## 1. Retorno dos Heróis

A documentação apresenta duas versões incompatíveis.

### Opção A — retorno direto ao Campo

> Pague 3 Faíscas para colocar o Herói da Reserva no Campo. Ele entra Cansado.

### Opção B — retorno à mão

> Pague 3 Faíscas para colocar o Herói da Reserva na sua mão.

A Opção A corresponde ao exemplo de Brasa e cria maior identidade mecânica, mas ainda precisa ser aprovada.

Também é necessário esclarecer:

- retornar conta como Invocar?
- efeitos “quando entra” são ativados?
- Veloz interfere no estado Cansado?
- preparar o Herói permite usá-lo no mesmo turno?

## 2. Resistente

A frase “ignora o primeiro dano recebido” não define quantidade nem frequência.

### Redação provisória

> Sempre que esta criatura receber dano, reduza esse dano em 1.

Avaliar em playtest se deve funcionar apenas uma vez por turno.

## 3. Curador

Os nomes **Enraizado** e **Ardente** aparecem em uma lista antiga, mas não possuem regras na v1.0.

Decidir entre remover, definir ou reservar esses efeitos.

## 4. Marcador versus efeito

Padrão recomendado:

| Marcador | Efeito |
|---|---|
| Gelo | Congelado |
| Teia | Preso |
| Rachadura | Quebrado |
| Chama + | Queimar |
| Muro + | Fortificado |
| Punho + | Imponente |
| Seta para baixo − | Fraco |
| Seta para cima + | Forte |
