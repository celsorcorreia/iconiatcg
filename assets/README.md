# Assets

Armazene aqui os recursos visuais permanentes do projeto.

## Estrutura recomendada

```text
assets/
├── cards/
├── icons/
│   ├── abilities/
│   ├── effects/
│   └── elements/
├── logos/
├── markers/
├── playmats/
└── references/
```

## Regras

- Não use URLs temporárias do Notion.
- Prefira SVG para símbolos e ícones.
- Mantenha fontes editáveis separadas das exportações.
- Use nomes em `kebab-case`.
- Inclua autoria e licença quando aplicável.
- Teste os ícones no tamanho real da carta.
