# Iconia TCG

Repositório de desenvolvimento de **Iconia: Bichos & Feitiços**, um jogo de cartas colecionáveis visual-first, com mínima dependência de texto, simples e acessível, ambientado em um mundo de bichos mágicos, Livros de Magia vivos e aventuras elementais.

> **Estado das regras:** Livro de Regras Simples v1.0 em validação.  
> **Sincronização inicial:** conteúdo organizado a partir do Notion em 18 de julho de 2026.  
> **Última verificação:** Regras principais revisadas no Notion em 18 de julho de 2026, às 16:50 (America/Sao_Paulo).

## Visão rápida

Cada jogador assume o papel de um **Invocador** escolhido por um Livro de Magia. Usando **Faíscas**, os jogadores invocam criaturas, lançam magias e manifestam itens para Explorar Iconia e reunir **Iconitos**.

O primeiro jogador a reunir **20 ou mais Iconitos** vence imediatamente.

## Parâmetros atuais

| Parâmetro | Regra |
|---|---|
| Tamanho do Livro de Magia | 30 cartas |
| Mão inicial | 7 cartas |
| Limite de mão | Sem limite |
| Cópias da mesma carta | Até 4 |
| Cópias de cada Herói | 1 |
| Iconitos iniciais | 0 |
| Faíscas iniciais | 0 |
| Acender | 1 carta por turno |
| Vitória | 20 ou mais Iconitos |
| Primeiro jogador | Não compra no primeiro turno |
| Formato físico | 63 mm × 88 mm |
| Público principal | 8–10 anos |
